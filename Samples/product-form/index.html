# Name Price Avalability

About project

This web-based application allows you to create new products and display them in a catalog table. You can easily enter the name and price of a product using the provided form, and indicate whether it is in stock by checking the corresponding checkbox. When you submit the form, a new product is created and added to the catalog table.

Each product in the table is assigned a unique number for easy identification. The table displays the product number, name, price, and availability. The availability is indicated by a badge, showing In Stock or Out Of Stock based on your selection.

If you want to start fresh and remove all the products from the table, simply click the Clear All button. This action will clear the table and reset the product numbering.

The project features a user-friendly interface with a navigation bar for easy access. You can learn more about the project by clicking the About link in the navigation bar, which opens a modal window with additional information.

Enjoy creating and managing your product catalog effortlessly with this Object-Oriented JavaScript application!
